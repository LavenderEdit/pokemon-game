# 🎮 Pokémon Battle Simulator (Pygame)

A retro Game Boy Advance (GBA) style Pokémon battle simulator built using Python, Pygame, and the PokeAPI.

## 🚀 Features
- 60 FPS optimized game loop.
- Dynamic data fetching using PokeAPI.
- Local GBA sprite and UI rendering.
- Smooth health bar animations.

## 📁 Project Structure
- `assets/`: Game art, UI textures, and fonts.
- `src/`: Python source code modules.

## 🛠️ Setup Instructions
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the game:
   ```bash
   python src/main.py
   ```
